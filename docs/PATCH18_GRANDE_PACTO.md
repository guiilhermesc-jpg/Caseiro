## Posicao oficial

Esta imagem e uma previa de expansao. Ela mostra o alvo visual e sistemico do Patch 18, mas nao deve ser tratada como recurso entregue na RV17.9.

## Escopo desejado do Patch 18

1. Base instalavel premium: PWA forte, preparo para pacote pesado e decisao tecnica sobre app instalado. **RV18.0 iniciou isto com o launcher.**
2. Portal publico: noticias, patch notes, personagens, conta, ranking, mapa e bestiario.
3. Rotas altas: caminhos de montanha, plataformas, portais e viagens com custo.
4. Primeiro grande calabouco: entrada marcante, progresso por alas, recompensa unica e boss de etapa.
5. Crescimento draconico: dormir, afinidade, treino, custo, estamina e preparo para voo.
6. Performance: imagens pesadas so entram se o cache, build e dispositivos-alvo aguentarem.

## Regra de ouro

O Patch 18 deve parecer maior sem virar promessa falsa. Cada screenshot oficial precisa apontar para cena, sistema, NPC, item, quest ou cronograma fechado.

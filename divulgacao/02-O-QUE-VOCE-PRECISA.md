## ✅ Checklist geral (vale para os dois projetos)

- [ ] **Conta pessoal no Facebook** (é a "chave" que administra tudo; ninguém vê que é você por trás).
- [ ] **Página de Facebook** do negócio (≠ perfil pessoal). Ex.: "STS Technology".
- [ ] **Perfil no Instagram** do negócio, configurado como **conta profissional/comercial**.
- [ ] **Instagram conectado à Página do Facebook** (para anunciar nos dois ao mesmo tempo).
- [ ] **Meta Business Suite / Gerenciador de Negócios** criado (organiza páginas, contas e cobrança).
- [ ] **Conta de Anúncios** criada dentro do Gerenciador.
- [ ] **Forma de pagamento** cadastrada (cartão de crédito; às vezes Pix/boleto).
- [ ] **WhatsApp Business** instalado e com mensagem de saudação configurada.
- [ ] **Criativos** prontos: pelo menos 3 imagens **ou** vídeos curtos por projeto.
- [ ] **Destino do clique** definido: WhatsApp e/ou site `ststechnology.com.br`.

## 🧱 Sobre o Pixel da Meta (vale entender)

O **Pixel** é um codizinho que você cola no seu site. Ele permite:

- **Medir** quantas pessoas que clicaram no anúncio realmente visitaram o site / pediram orçamento.
- **Reimpactar** ("remarketing") quem visitou mas não fechou — mostrando o anúncio de novo depois.
- Deixar a Meta **otimizar para conversão**, não só para clique.

Para um site simples, dá para instalar via **Meta Pixel** (código no `<head>`) ou por plugins/integração.

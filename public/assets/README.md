# Assets de imagem (ilustrações)

Coloque aqui os PNGs gerados (veja `docs/ARTE.md` e `docs/PROMPTS-ARTE.md`):

- `itens/<id>.png`        → ícones dos itens (id igual ao de `src/data/itens.js`)
- `personagens/<id>.png`  → `jogador`, `mae`, `filho`, `avo`

**Depois de salvar um arquivo**, registre o `id` em `src/data/sprites.js`.
O jogo passa a usar a ilustração no lugar do emoji automaticamente.

Especificação: PNG **transparente**, **512×512**, estilo **graphic novel** (ver `ARTE.md`).
